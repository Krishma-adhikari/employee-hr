# Employee HR Analytics Dashboard

An interactive HR analytics dashboard built with React, TypeScript, and Vite.

## Features
- Interactive HR dashboard
- Employee statistics
- Attrition visualizations
- Responsive design

## Live Demo
https://employee-hr-krishma.vercel.app/

## Related Project
The data analysis, cleaning, EDA, and machine learning work are available in a separate repository:
https://github.com/Krishma-adhikari/employee-attrition-analysis

## Note
The live demo showcases the frontend dashboard.

AI-powered features (risk prediction and HR assistant) are implemented in the codebase but are disabled in the live deployment because they require an Express backend and Gemini API server, which are not deployed on Vercel.
